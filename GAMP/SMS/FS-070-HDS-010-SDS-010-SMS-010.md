# FS-070-HDS-010-SDS-010-SMS-010 Foreground Supervisor

Implements event selection, exact-child process checks, bounded functional
health, candidate promotion, verified rollback, guest-shutdown jitter/restart,
explicit rollout signal handling, QMP powerdown best effort, and terminal phase
reporting. Eligible primary events call the FS-160 module before activation;
force-candidate and recovery events never call it.
