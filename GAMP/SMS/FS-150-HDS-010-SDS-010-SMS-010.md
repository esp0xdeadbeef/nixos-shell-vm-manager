# FS-150-HDS-010-SDS-010-SMS-010 Console Adapter

`modules/default.nix` emits the enabled flag, stable socket path, and session
name into each root-owned instance configuration. The supervisor module in
`scripts/nixos-shell-vm-manager.sh` creates the tmux session, runs the selected
runner in its pane, checks that exact pane process, captures natural exit
status, and cleans the session during stop or image replacement.
