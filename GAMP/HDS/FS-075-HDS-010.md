# FS-075-HDS-010 Host-session Start-authority State

Volatile per-VM authority state records a monotonic intent epoch and a typed
stop marker. An operator service stop records `explicit-stop`; a configured
carrier-down transition records the distinct automatic `carrier-down` reason.
Both advance the epoch before terminating the supervisor. Carrier-up may clear
only `carrier-down`; it preserves `explicit-stop`. Every start transition checks
the current marker while holding the lifecycle lock. Candidate admission never
edits authority state.
