# FS-120-HDS-010-SDS-010 NixOS Module Interface

The public option namespace is `services.nixosShellVmManager`. Each instance
requires `image` and `healthCheck.command`, and exposes activation, storage,
health bounds, runner arguments, local flake attribute, and optional pin-refresh
flake reference and build attribute. Enabling `activation.refreshPins` without
both refresh inputs is rejected. Generated root-owned configuration contains
only resolved values. A safe relative runner path supports
intentional configuration-name/guest-name compatibility aliases without
encoding a consumer repository assumption.
