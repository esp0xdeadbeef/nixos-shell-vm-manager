# FS-040-HDS-010-SDS-010 Atomic Image-state Registry

Registry mutations hold the lifecycle lock, validate an executable
the configured safe relative runner path (by default `bin/run-<vm>-vm`), create
an immutable GC root, write a complete temporary JSON
document with `jq`, fsync by rename semantics, and atomically replace state.
Readers never source provenance as shell. Unreferenced GC roots are pruned only
after a committed state update.
