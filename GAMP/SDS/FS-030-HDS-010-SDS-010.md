# FS-030-HDS-010-SDS-010 Immutable Local Candidate Pipeline

`vm-update` validates root authority, instance membership, source directory,
`flake.nix`, and `flake.lock`; records the authority epoch; acquires per-VM and
shared build locks; archives with lock updates and lock writes disabled; builds
the configured attribute from the archive store path; then calls the common
registrar. Only after admission does it recheck authority and request rollout.
